## 03: Linear Algebra - review

1. 概念：
   - matrix: rectangular array of numbers: rows x columns
   - element: i -> ith row, j -> jth column
   - vector: a nx1 matrix
2. 操作：
   - 加和: 需要相同的维，才能元素级别的相加减。
   - 标量乘积
   - 混合运算
